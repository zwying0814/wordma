# 关于我们

欢迎来到 Wordma，一个现代化的多主题博客系统。

## 项目特色

Wordma 采用 monorepo 架构，支持多个主题，让你可以轻松切换不同的设计风格。

## 技术栈

- **Astro**: 现代静态站点生成器
- **TypeScript**: 类型安全的 JavaScript
- **PNPM**: 高效的包管理器
- **MDX**: 支持 JSX 的 Markdown
